## ⚡ Tech Stack

- 🖥️ **Vue.js 3**
  - Composition API
  - Script Setup Syntax
  - TypeScript Support

- 🎨 **Tailwind CSS**
  - Custom Design System
  - Responsive Utilities
  - Dark Mode Support

- 📦 **Pinia**
  - Module-based Stores
  - DevTools Integration
  - Composition API Support

- 🛣️ **Vue Router**
  - Navigation Guards
  - Route Meta Fields
  - Dynamic Routing
